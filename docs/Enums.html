  • DiscordEndpoint

    This enum controls the interface with the Discord REST API.

    It defines several endpoint cases which are then used by the various static methods defined on this enum. The cases themselves are of little use.

    The methods all take a DiscordToken which is used for authentication with the REST API. GET methods also take a callback.

    All requests through this enum are rate limited.
