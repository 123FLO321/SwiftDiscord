DiscordVoiceEngine

public final class DiscordVoiceEngine : DiscordEngine, DiscordVoiceEngineSpec

A subclass of DiscordEngine that provides functionality for voice communication.

Discord uses encrypted OPUS encoded voice packets. The engine is responsible for encyrpting/decrypting voice packets that are sent and received.

    Takes a process that outputs random audio data, and sends it to a hidden FFmpeg process that turns the data into raw PCM.

youtube.launchPath = "/usr/local/bin/youtube-dl"
youtube.arguments = ["-f", "bestaudio", "-q", "-o", "-", link]
